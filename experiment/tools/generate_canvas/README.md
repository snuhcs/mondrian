# Canvas Generator 

Generate packed canvases and save at `{log_dir}/canvas/{canvas_index}.jpg`

## Usage
`main.py [-h] --log-dir LOG_DIR --video_path VIDEO_PATH`

  - `-h`, `--help` : show this help message and exit
  - `--log-dir LOG_DIR`, `-d LOG_DIR` : Path to log directory
  - `--video_path VIDEO_PATH`, `-v VIDEO_PATH` : Path to video file
