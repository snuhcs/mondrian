# Mondrian


## About The Project
This project enables effective multi-stream video analytics via spatio-temporal packing.

**Paper**: [Mondrian: On-device high-performance video analytics with compressive packed inference](https://arxiv.org/abs/2403.07598)


## Directory Structure
* [`android/`](android/README.md): Mondrian android runtime
* [`offline/`](offline/README.md): Mondrian offline stage: train ROI scale estimator
